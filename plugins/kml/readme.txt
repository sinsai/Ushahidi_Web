== Installation ==
1. Copy the entire /kml/ directory into your /plugins/ directory.
2. Activate the plugin through the administrator Addons interface at /admin/addons/plugins/
3. Navigate to http://www.yourwebsite.com/kml to download the KML (actually a compressed KMZ file, by recent 1000 items)
4. Optionally navigate  http://www.youywebsite.com/kml?l=<limit>&cat=<category ID>
5. Options for CDN, site url, balloon content, caching, etc. are available at /kml/config/kml.php
